# nextjs-tutorial

Next.js Learn の Dashboard App チュートリアルを進めるための個人学習用リポジトリです。

## Overview

このリポジトリでは、Next.js App Router を用いたダッシュボードアプリの構築を通して、以下を学習します。

- Routing
- Layout
- Server Components
- Data Fetching
- Database Setup
